# Chibitami.net-reborn
Old chibitami.net/works site recreation with flash content from Yu-ki


if you know Yu-ki (chibitami) you know about his site, yup, it dissapeared from the internet
along with all of his flash games and you're thinking you miss the site right?

Well... i introduce to you "Chibitami Reborn"!
chibitami Reborn is a total recreation of the old yu-ki's site with all the assets the site had
and of course, all the online games are now ready to be played thanks to Ruffle Emulator

this project doesnt limit on only the online games yu-ki released to be played online. but in the next few days, im gonna
add too the exclusive games from Meiso Potage from Yu-ki.

just like with Meiso Potage Flash Archive, this project was totally recreated from my private server, so all the main stuff is ready for this repository.
